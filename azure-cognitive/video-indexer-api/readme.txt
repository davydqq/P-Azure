Required Azure Resources
In order to run these demos, you will need to create:
An Azure Storage Account
A Video Indexer Account

C# Demos
The VideoIndexerApiClient.ConsoleApp application contains code to run all the demos
Set the storage account and video indexer account details
When required, set the media asset id
Comment in the relevant code
Build and run the demos


Python Demos
The PyFiles folder contains the python files for the demos
The VideoIndexerApiClient contains the Visual Studio solution.
Set the storage account and video indexer account details
When required, set the media asset id
Run the python files.
